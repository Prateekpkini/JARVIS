# 🎙️ J.A.R.V.I.S. Voice Assistant 

Hey there! 👋 Welcome to my personal J.A.R.V.I.S. project. 

As an AIML engineering student, I spend a lot of time working with data and models, but I wanted to build something I could actually *talk* to. This repository houses the evolution of my custom voice assistant—designed to fetch real-time news, manage tasks, and interact naturally.

It's been a fun journey of iterating through different versions (yes, I use the Iron Man "Mark" naming system 🚀), upgrading from a simple terminal script to a fully functional GUI with a virtual avatar.

## ✨ What Can JARVIS Do?
Right now, the assistant is capable of:
* **Real-Time News:** Fetches the latest headlines on command.
* **Voice Interaction:** Listens to voice inputs and responds with text-to-speech.
* **Gemini AI Integration:** Powered by generative AI for smarter, more context-aware responses.
* **Interactive GUI:** Features a dedicated user interface complete with an animated avatar (`avatar.gif`).

## 📂 Project Structure & Evolution
You'll notice a few different files and folders in here. I like keeping a record of my progress:

* **`JARVIS_MARK_21_Beta/`**: The latest and greatest. Currently squashing bugs and testing new features.
* **`JARVIS_MARK_20_Official/`**: The last stable release.
* **`jarvis_day2(Terminal upgrade).py`**: A throwback to the earlier, terminal-only days.
* **`jarvis_gui_avatar.py`**: The core script handling the graphical interface and animations.
* **`TEST-your_env_for_gemini.py`**: Testing environment for the Gemini API integration.
